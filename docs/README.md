# GitHub Pages 视频页面

此文件夹包含用于在GitHub Pages上显示视频的HTML页面。

## 设置步骤

1. 在GitHub仓库中，进入 **Settings** (设置)
2. 在左侧菜单中找到 **Pages** (页面)
3. 在 **Source** (源) 部分，选择 **Deploy from a branch** (从分支部署)
4. 选择 **main** (或 **master**) 分支
5. 选择 **/docs** 文件夹
6. 点击 **Save** (保存)

等待几分钟后，GitHub Pages会自动部署，视频页面将可以通过以下URL访问：
`https://YOUR_USERNAME.github.io/2dof_tvb_read/video.html`

请将 `YOUR_USERNAME` 替换为您的GitHub用户名。
